Konteks Penelitian:

Penelitian ini diajukan untuk ajang kompetisi IYSA 2026. Pengembangan model klasifikasi awan menggunakan algoritma Convolutional Neural Network (CNN) dengan variasi dataset untuk meningkatkan akurasi dalam berbagai kondisi pencahayaan.

Catatan: prediksi ini bersifat pendekatan sederhana berdasarkan fitur visual awan dari tanah, dan tidak menggantikan sistem meteorologi resmi.
